== Description ==
This plugin provides advanced search filters for WooCommerce products. **Requires WooCommerce to be active**.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/search-advanced-filter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Make sure WooCommerce is installed and activated.

== Frequently Asked Questions ==
= Do I need WooCommerce for this plugin to work? =
Yes, WooCommerce must be installed and activated for this plugin to function properly.
